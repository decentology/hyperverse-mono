# Welcome!

We have integrated Storybook with **Builder Kit (SafuuX)** in an effort to help users interact with the different components in this module.

Feel free to browse our example stories from our Hyperverse modules now by navigating to them in the left sidebar under **Components**.
Every story includes a demo and a docs page.
You can view each demo under the **Canvas** tab and the docs under the **Docs** tab.


<div className="subheading">About Builder Kit</div>

The **Builder Kit** (formerly known as DappStarter) is a full stack development environment for blockchains.
It supports an integrated front end user experience in your choice of client side framework and different smart contract features.

**Blockchains:** Blockchains are the underlying technology for the decentralized web.
There are numerous public and private blockchains and Builder Kit will support many of them soon.

**Languages:** Different blockchains use different languages. Some blockchains use one language while others use several languages.

**Frameworks:** Includes a starter client app (optional; can be replaced) using Tailwind CSS and JavaScript. You choose!

**Smart Contract Features:** Users will also be able to add differentfeatures that will give you the customized smart contract needed for your application.

**Dapp Customization:** Tailor your blockchain app to fit your brand with your own Dapp name.

<div className="subheading">About Storybook</div>

Storybook helps users with their UI development, testing, and documentation.

Each story includes a demo and a docs page. Our demonstrations
simulate the user's interaction with each component. This is done by providing props for the initial state.

While we have provided a set of stories for you to get started with, you are welcome to create your own!
Navigate to the **Stories** section on the sidebar for more information on how to create them.
